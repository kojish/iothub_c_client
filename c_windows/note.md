# Libraries
The following packages are required to compile:  
- Microsoft.Azure.IoTHub.IoTHubClient
- Microsoft.Azure.C.SharedUtility
- Microsoft.Azure.IoTHub.MqttTransport
- Microsoft.Azure.umqtt
